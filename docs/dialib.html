DiaLib

DiaLib is a powerful GUI library for Unity that simplifies the process of creating and managing GUI elements. It provides a set of classes and methods that enable you to build interactive and user-friendly interfaces for your Unity projects.

How to Use DiaLib:

To use DiaLib in your Unity project, follow these steps:

  1. Download the DiaLib package from the official GitHub repository.
  2. Import the DiaLib package into your Unity project.
  3. Create an instance of the GUIManager class to manage your GUI elements.
  4. Add GUI elements such as buttons, toggles, and text fields using the GUIManager's AddButton, AddToggle, and AddTextField methods.
  5. Customize the appearance and behavior of the GUI elements as needed.
  6. Run your Unity project to see the GUI in action.

Example Code:


    using UnityEngine;
    using DiaLib;

    public class MyGUIController : MonoBehaviour
    {
        private GUIManager guiManager;

        private void Start()
        {
            guiManager = new GUIManager(Color.white, new Color(0.2f, 0.2f, 0.2f, 0.8f), "My GUI Window");
            guiManager.AddButton("Click Me", (label) => Debug.Log("Button Clicked: " + label));
            guiManager.AddToggle("Enable Feature", true, (value) => Debug.Log("Toggle Value Changed: " + value));
            guiManager.AddTextField("Enter Name:", "", (text) => Debug.Log("Text Changed: " + text));
        }

        private void OnGUI()
        {
            guiManager.DrawGUI();
        }
    }
