DiaLib

DiaLib is a simple GUI library for Unity that makes it easier to create and manage GUI elements for your projects. It provides essential classes to build interactive and user-friendly interfaces without the need for complex code.

Key features of DiaLib:

  • Easy-to-use API for creating GUI elements.
  • Allows customizing GUI element appearance, colors, and positions.
  • Simplifies handling user interactions and events.
  • Supports buttons, toggle switches, and text fields.
  • Provides a clear and straightforward interface for GUI management.

GUIManager

The `GUIManager` is the core component of DiaLib. It manages the GUI elements and handles drawing the GUI window along with user interactions.

Properties:

  • GUIColor: Gets or sets the color of the GUI elements.
  • BackgroundColor: Gets or sets the background color of the GUI window.
  • Title: Gets or sets the title of the GUI window.

Methods:

  • AddButton: Adds a button to the GUI window.
  • AddToggle: Adds a toggle switch to the GUI window.
  • AddTextField: Adds a text field to the GUI window.

Example:


// Create a new GUIManager
GUIManager guiManager = new GUIManager();

// Set the GUI color and background color
guiManager.GUIColor = Color.white;
guiManager.BackgroundColor = new Color(0.2f, 0.2f, 0.2f, 0.8f);

// Set the GUI window title
guiManager.Title = "My GUI Window";

// Add a button to the GUI window
guiManager.AddButton("Click Me", (label) => Debug.Log("Button Clicked: " + label));

EasyGUI

The `EasyGUI` is a static utility class that simplifies working with `GUIManager`. It provides a user-friendly interface to create GUI elements with less code.

Properties:

  • GUIColor: Gets or sets the color of the GUI elements.
  • BackgroundColor: Gets or sets the background color of the GUI window.
  • Title: Gets or sets the title of the GUI window.

Methods:

  • BeginWindow: Begins a GUI window with custom content.
  • Button: Adds a button to the GUI window with a custom onClick action.
  • Toggle: Adds a toggle switch to the GUI window with a custom onValueChanged action.
  • TextField: Adds a text field to the GUI window with a custom onTextChanged action.

Example:


// Set the GUI color and background color using EasyGUI
EasyGUI.GUIColor = Color.white;
EasyGUI.BackgroundColor = new Color(0.2f, 0.2f, 0.2f, 0.8f);

// Set the GUI window title using EasyGUI
EasyGUI.Title = "My GUI Window";

// Add a button to the GUI window using EasyGUI
EasyGUI.Button("Click Me", () => Debug.Log("Button Clicked!"));

Parts of DiaLib

In addition to the `GUIManager` and `EasyGUI`, DiaLib includes other essential components:

GUIButton

The `GUIButton` represents a button element in the GUI. It allows you to create clickable buttons with custom labels and execute actions when the button is clicked.

GUIToggle

The `GUIToggle` represents a toggle element in the GUI. It allows you to create toggleable switches with custom labels and execute actions when the toggle value is changed.

GUITextField

The `GUITextField` represents a text field element in the GUI. It allows you to create input fields for users to enter text and execute actions when the text is changed.

Conclusion

With DiaLib, you have all the necessary tools to create a rich and interactive GUI for your Unity project. The library simplifies the process of creating GUI elements and managing their interactions, allowing you to focus on building a user-friendly interface.
