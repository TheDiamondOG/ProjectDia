GUIManager

The `GUIManager` class is the core component of DiaLib. It acts as a manager for GUI elements and is responsible for drawing the GUI window and handling user interactions.

Properties:

  • GUIColor: Gets or sets the color of the GUI elements.
  • BackgroundColor: Gets or sets the background color of the GUI window.
  • Title: Gets or sets the title of the GUI window.

Methods:

  • AddButton: Adds a button to the GUI window.
  • AddToggle: Adds a toggle switch to the GUI window.
  • AddTextField: Adds a text field to the GUI window.

Example:


            GUIManager guiManager = new GUIManager();
            guiManager.GUIColor = Color.white;
            guiManager.BackgroundColor = new Color(0.2f, 0.2f, 0.2f, 0.8f);
            guiManager.Title = "My GUI Window";
            guiManager.AddButton("Click Me", (label) => Debug.Log("Button Clicked: " + label));
