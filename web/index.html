DiaLib

DiaLib is a powerful GUI library for Unity that simplifies the process of creating and managing GUI elements. It provides a set of classes and methods that enable you to build interactive and user-friendly interfaces for your Unity projects.

GUIManager

The `GUIManager` class is the core component of DiaLib. It acts as a manager for GUI elements and is responsible for drawing the GUI window and handling user interactions.

Properties:

  • GUIColor: Gets or sets the color of the GUI elements.
  • BackgroundColor: Gets or sets the background color of the GUI window.
  • Title: Gets or sets the title of the GUI window.

Methods:

  • AddButton: Adds a button to the GUI window.
  • AddToggle: Adds a toggle switch to the GUI window.
  • AddTextField: Adds a text field to the GUI window.

Example:


    GUIManager guiManager = new GUIManager();
    guiManager.GUIColor = Color.white;
    guiManager.BackgroundColor = new Color(0.2f, 0.2f, 0.2f, 0.8f);
    guiManager.Title = "My GUI Window";
    guiManager.AddButton("Click Me", (label) => Debug.Log("Button Clicked: " + label));

EasyGUI

The `EasyGUI` class is a static utility class that provides a simplified interface for working with the `GUIManager`. It encapsulates common GUI elements and layout functionalities, making it easier to create GUIs with less code.

Properties:

  • GUIColor: Gets or sets the color of the GUI elements.
  • BackgroundColor: Gets or sets the background color of the GUI window.
  • Title: Gets or sets the title of the GUI window.

Methods:

  • BeginWindow: Begins a GUI window with a custom content section.
  • Button: Adds a button to the GUI window with custom onClick action.
  • Toggle: Adds a toggle switch to the GUI window with custom onValueChanged action.
  • TextField: Adds a text field to the GUI window with custom onTextChanged action.

Example:


    EasyGUI.GUIColor = Color.white;
    EasyGUI.BackgroundColor = new Color(0.2f, 0.2f, 0.2f, 0.8f);
    EasyGUI.Title = "My GUI Window";
    EasyGUI.Button("Click Me", () => Debug.Log("Button Clicked!"));

Parts of DiaLib

In addition to the `GUIManager` and `EasyGUI` classes, DiaLib includes other useful parts to enhance your GUI functionality.

GUIButton

The `GUIButton` class represents a button element in the GUI. It allows you to create clickable buttons with custom labels and execute actions when the button is clicked.

Properties:

  • Label: Gets or sets the label text displayed on the button.
  • OnClick: Gets or sets the action to be executed when the button is clicked.

Example:


    GUIButton myButton = new GUIButton("Click Me!", () => Debug.Log("Button Clicked!"));

GUIToggle

The `GUIToggle` class represents a toggle element in the GUI. It allows you to create toggleable switches with custom labels and execute actions when the toggle value is changed.

Properties:

  • Label: Gets or sets the label text displayed next to the toggle.
  • Value: Gets or sets the current value of the toggle.
  • OnValueChanged: Gets or sets the action to be executed when the toggle value is changed.

Example:


    GUIToggle myToggle = new GUIToggle("Enable Feature", true, (value) => Debug.Log("Toggle Value Changed: " + value));

GUITextField

The `GUITextField` class represents a text field element in the GUI. It allows you to create input fields for users to enter text and execute actions when the text is changed.

Properties:

  • Label: Gets or sets the label text displayed above the text field.
  • Text: Gets or sets the current text value in the text field.
  • OnTextChanged: Gets or sets the action to be executed when the text in the field is changed.

Example:


    GUITextField myTextField = new GUITextField("Enter Name:", "", (text) => Debug.Log("Text Changed: " + text));

Conclusion

With the `GUIManager`, `EasyGUI`, and other parts of DiaLib, you have all the necessary tools to create a rich and interactive GUI for your Unity project. The library simplifies the process of creating GUI elements and managing their interactions, allowing you to focus on building a user-friendly interface.
